atlas.nvim — colourscheme mockup

VS Code Dark+ role semantics rendered from each theme’s own colors.sh. Every role is derived from the existing 16-colour palette by mixing toward TEXT, so one expression works on dark and light themes alike and every future theme gets a colourscheme for free.
